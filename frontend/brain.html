The Tab OKF extension

Before anything is shared, it's yours. The extension saves a summary of each tab into a private local database on your own machine — so you can finally close tabs instead of letting them stack up. Press Ctrl+Shift+S on any tab: banked and closed, a badge counts your day. Then post the ones worth sharing up to the hivemind.

  1. Download the extension and unzip it.
  2. Open chrome://extensions, turn on Developer mode.
  3. Load unpacked → pick the unzipped extension/ folder.
  4. It banks to a local capture server on your own machine — see the included README to run it.
