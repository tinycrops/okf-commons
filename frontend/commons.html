The full tool

The Tab OKF extension

The web form above is the easy path. The extension is the real one: a one-key reward loop in your browser. Press Ctrl+Shift+S on any tab and it's banked into your own OKF memory and closed — a badge counts how many you've banked today. Ctrl+Shift+E opens a window to add why first.

  1. Download the extension and unzip it.
  2. Open chrome://extensions, turn on Developer mode.
  3. Load unpacked → pick the unzipped extension/ folder.
  4. It banks to a local capture server on your own machine — see the included README to run it.

The extension banks to your machine, privately. The commons on this page is the shareable layer on top.
