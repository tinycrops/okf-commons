◆ OKF MINI — THE 2-MINUTE EXTENSION

One button. Summarize the tab,
save it, close it.

The friend-sized version of the OKF memory tool. No local server, no setup to learn — click once and the page is compressed into our shared hivemind and the tab closes. The reward is the close, not the pile of 100 open tabs.

Install in 2 minutes Read the raw spec →
What it does
1 · You're on a keeper. An article, a repo, a thread you'd leave open "for later."
2 · You click the button (or press Ctrl+Shift+S).
3 · The backend summarizes it into a concept — title, summary, why it mattered, tags — and folds it into the shared hivemind.
4 · The tab closes. A ✓ flashes. It's remembered; your screen is clean.

Deliberately tiny: two permissions, no page-scraping. The backend fetches the URL itself, so the extension never touches the content of your pages. Works for any public page (login-walled / heavy-JS pages are what the full extension is for).

The contract
Everything routes through one call: POST https://ath-ms-7a73.tail6017fa.ts.net/okf/api/contribute with { url, contributor?, intent? } — returns 202 instantly and summarizes in the background. Banked concepts appear in the hivemind.
The three files

Make a folder okf-mini/ and drop these in. Or hand this page to your coding agent.

manifest.json

{
  "manifest_version": 3,
  "name": "OKF Mini — Summarize & Close",
  "version": "1.0.0",
  "description": "Bank the current tab into your shared OKF memory, then close it.",
  "permissions": ["activeTab", "storage"],
  "host_permissions": ["https://ath-ms-7a73.tail6017fa.ts.net/*"],
  "background": { "service_worker": "background.js" },
  "action": { "default_title": "Bank & close this tab (Ctrl+Shift+S)" },
  "options_page": "options.html",
  "commands": {
    "bank-and-close": {
      "suggested_key": { "default": "Ctrl+Shift+S" },
      "description": "Bank & close the current tab"
    }
  }
}

background.js

const BACKEND = "https://ath-ms-7a73.tail6017fa.ts.net/okf";

async function badge(text, color) {
  await chrome.action.setBadgeBackgroundColor({ color: color || "#c9821f" });
  await chrome.action.setBadgeText({ text });
}

async function bank(tab) {
  if (!tab || !tab.url || !/^https?:/i.test(tab.url)) return;
  const { contributor } = await chrome.storage.sync.get("contributor");
  await badge("…");
  try {
    const r = await fetch(BACKEND + "/api/contribute", {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({ url: tab.url, contributor: contributor || "" }),
    });
    if (r.ok) {                       // 202 — accepted
      await badge("✓", "#3f9e57");
      await chrome.tabs.remove(tab.id);
    } else {
      await badge("!", "#c2533f");     // rate-limited — slow down, tab stays
    }
  } catch (e) {
    await badge("x", "#c2533f");        // backend asleep — nothing lost, tab stays
  }
  setTimeout(() => badge(""), 2500);
}

chrome.action.onClicked.addListener(bank);
chrome.commands.onCommand.addListener(async (cmd) => {
  if (cmd !== "bank-and-close") return;
  const [tab] = await chrome.tabs.query({ active: true, currentWindow: true });
  bank(tab);
});

Install

Two minutes, Chrome / Edge / Brave

  1. Make a folder okf-mini/ and save the three files above inside it.
  2. Open chrome://extensions.
  3. Turn on Developer mode (top-right toggle).
  4. Click Load unpacked → choose the okf-mini/ folder.
  5. Optional: open the extension's options and set your name.
  6. Pin the icon. On any page, click it (or Ctrl+Shift+S) to bank & close.

Everything you bank shows up in the shared hivemind. The backend lives on the maintainer's machine — if it's asleep you'll get an x badge and the tab simply stays open.
